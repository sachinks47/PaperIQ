📄 PaperIQ – AI Academic Writing Analyzer

PaperIQ is an AI-powered academic writing evaluation tool developed during Infosys Springboard Virtual Internship 6.0.

It analyzes research documents and provides:

• Language Quality Score  
• Coherence Score  
• Reasoning Strength  
• Lexical Sophistication  
• Readability Index  
• Smart Section Summaries  
• PDF Downloadable Report
